TicTacToe

Der zweck dieses Programmes ist das ich lerne wie man die Basics von Rust benutzt und verstehe.

Startanletung

Am anfang des Programmes wird ein Start-Bildschirm angezeigt der dich fragt ob du das Spiel starten möchtest oder es beenden möchtest.
(Q = Beenden, ENTER = Spiel beginnen)
 
Bediengungen 
Wenn du Enter drückst (das Spiel startest) wird das TicTacToe feld angezeigt und du Kannst ein Feld wählen. Am ende des Spieles wirst du gefragt ob du nochmal spielen möchtest, oder das Spiel beenden möchtest.
